Why rotate shoes?

  • Injury prevention: Alternating midsoles reduces repetitive stress on joints.
  • Durability: Letting foam decompress between runs extends life.
  • Performance: Dedicated shoes for speed vs easy runs maximise results.
  • Enjoyment: A different feel underfoot can boost motivation.

Essential roles in a rotation

1. Daily Trainer

The workhorse. Durable, comfortable, and versatile. Examples:

  • Nike Pegasus 41
  • Saucony Ride 17

2. Long-Run Cushion

Max cushioning for Sunday long runs and recovery days. Examples:

  • Brooks Glycerin 21
  • New Balance 1080 v13

3. Speed / Workout Shoe

Lighter, snappier trainers for intervals and tempo runs. Examples:

  • Saucony Endorphin Speed 4
  • ASICS Novablast 4

4. Race Day Shoe

Usually carbon-plated with responsive foams. Expensive, but save for races and key tune-ups. Examples:

  • Nike Alphafly 3
  • Adidas Adios Pro 3

5. Trail / Off-Road Option

If you run on mud, grass, or hills, add a trail shoe for grip and protection. Examples:

  • Salomon Speedcross 6
